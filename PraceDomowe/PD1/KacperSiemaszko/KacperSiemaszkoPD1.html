Wprowadzenie

Tematem, ktrym chciabym si dzisiaj zaj jest przebieg wojny handlowej pomidzy Chinami, a Stanami Zjednoczonymi. Na podstawie raportu Global Financial Stability Report October 2018: A Decade after the Global Financial Crisis: Are We Safer? (rdo: https://www.imf.org/en/Publications/GFSR/Issues/2018/09/25/Global-Financial-Stability-Report-October-2018) zestawiem ze sob zmiany cen aktyw firm zwizanych z handlem midzynarodowym. Grupy, ktre zostay wyrnione to:

  • Firmy, ktrych sprzedaj duo towarw na rynku chiskim,
  • Sektory, ktre maj duy wpyw chiskich towarw,
  • Firmy, ktre zajmuj si handlem midzynarodowym.

Dwie pierwsze grupy pozwalaj nam na okrelenie jak wojna handlowa wpywa na import/export pomidzy Stanami, a Chinami. Ostatnia suy jako grupa kontrolna, ktra pozwoli przedstawi czciowo wpyw trendw na rynku wiatowym. Oprcz samych zmian wartoci kapitau, pionowymi liniami oznaczone s wydarzenia, ktre dotychczas wpyny na przebieg wojny handlowej.

Przystosowanie danych

data <- read.csv(file="china_US.csv")

data$date <- as.Date(data$date, format = "%m/%d/%Y")
data <- melt(data, id = "date", measure = colnames(data)[2:4])
data$variable <- stri_replace_all(data$variable, " ", regex = "\\.")

Kod generowania wizualizacji

ggplot(data = data, aes(x=data$date, y = data$value, group = data$variable, color = data$variable)) +
  geom_line(size=1) +
  geom_vline(xintercept = as.Date("2018-07-06"), alpha=0.5, size = 1.5, color = "red") +
  geom_vline(xintercept = c(as.Date("2018-05-20"), as.Date("2018-06-12"), as.Date("2018-08-14")), alpha = 0.5) +
  scale_y_continuous(limits = c(min(data$value),max(data$value))) +
  scale_x_date(date_breaks = "1 month") +
  scale_color_manual(values = brewer.pal(3, "Accent")) + 
  xlab("date") +
  ylab("equity performace") +
  labs(title = "Impact of U.S.-China Trade Tensions on Asset Prices\n") + 
  theme_minimal() +
  theme(legend.title = element_blank(), plot.title = element_text(hjust = 0.5), legend.position = "bottom")

Wydarzenia, ktre uksztatoway przebieg wojny handlowej:

  • 20 maja - Zaagodzenie stosunkw, Chiny zgadzaj si na kupno wikszej iloci dbr ze Stanw
  • 12 czerwca - Rozmowy pokojowe pomidzy USA, a Kore Pnocn
  • 6 lipca - Oficjalny pocztek wojny handlowej - naoenie sankcji na Chiny przez Stany Zjednoczone
  • 20 sierpnia - Chiny zakadaj spraw przeciwko sankcjom USA do wiatowej Organizacji Handlu
